## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github

## Projeto

O Habits é um app para ajudar a rastrear os hábitos, cadastrando o dia atual para eventual controle.
